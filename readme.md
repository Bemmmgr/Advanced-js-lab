# Advanced JS Lab

A personal lab to strengthen my JavaScript skills.

## Structure

- `leetcodeJS-notes/`  
  JavaScript solutions and notes for LeetCode problems.

- `vanillaJS-snippets/`

- Get comfortable with JS syntax, data structures and algorithms.

- Keep a visible learning track on GitHub.
